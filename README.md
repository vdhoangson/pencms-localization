# PenCMS Localization

This is a localization package for PenCMS. You can easily integrate this package with any Laravel based project.

## Installation

Install it via Composer. Run this command in your terminal from your project directory:

```sh
composer require vdhoangson/pencms-localization
```

Wait for a while, Composer will automatically install in your project.
